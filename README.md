# Smart Gastos 💰📊

Aplicação web desenvolvida inteiramente com **React** e **TypeScript**, focada em **monitoramento e controle de gastos pessoais**.

O projeto permite registrar despesas, assinaturas recorrentes e visualizar tudo através de dashboards modernos e interativos.

🔗 Acesse: [https://gastos.vlopinhos.dev/]

## 📌 Sobre o Projeto

O sistema foi criado para oferecer uma visão clara e organizada da vida financeira do usuário.

Com ele é possível:

- 💸 Registrar gastos  
- 🔁 Cadastrar assinaturas recorrentes  
- 🚨 Receber alertas importantes  
- 📊 Visualizar dashboards animados e detalhados  

## 🎨 Interface e Experiência

Um dos grandes diferenciais do projeto é o cuidado com a interface.

Foi utilizado o **Mantine** como framework de UI, garantindo:

- 🎯 Componentes modernos  
- ✨ Animações suaves  
- 📱 Layout responsivo  
- 🎨 Design limpo e profissional  

## 🛠 Tecnologias Utilizadas

- React  
- TypeScript  
- Mantine (UI Framework)  

## 🎯 Objetivo

O foco principal foi:

- Criar uma aplicação robusta utilizando React + TypeScript  
- Trabalhar organização de estado e componentes  
- Desenvolver dashboards visuais e interativos  
- Aplicar boas práticas de UI/UX  

Projeto voltado para organização financeira com forte apelo visual e experiência moderna.
